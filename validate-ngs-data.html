Section 3 Validate NGS data

3.1 Step 1

  • Use woofr::validation_outputs to generate a TSV file with the following columns:
    • col1 - type of file
      1. VCF
      2. BAM
      3. FASTQ
      4. Other
    • col2 - path to file

3.2 Step 2

  • Now we can iterate through each row of the TSV according to its type:
    • VCF:
      • head
      • vcf-validator
    • BAM:
      • samtools quickcheck
      • GATK ValidateSamFile link1, link2
      • UMich BamUtil validate
    • FASTQ:
      • fqtools validate
